Place particles according to a Halton sequence from 0 to L (the box length) and uses a rejection radius to avoid placing particles to close to each other.

Parameters

  • bases (array) – Array of 3 ints each of which is a base for the Halton sequence. Defualt: bases = np.array([2,3,5])

  • r_reject (float) – Value of rejection radius.
