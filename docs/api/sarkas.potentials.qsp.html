sarkas.potentials.qsp module¶

Module for handling the Quantum Statistical Potential.

Note

Notice that in Ref. [Hansen1981] the DeBroglie wavelength is defined as

\[\lambda_{ee} = \dfrac{\hbar}{\sqrt{2 \pi \mu_{ee} k_{B} T} },\]

while in statistical physics textbooks and Ref. [Glosli2008] is defined as

\[\lambda_{ee} = \dfrac{h}{\sqrt{2 \pi \mu_{ee} k_{B} T} },\]

References

Hansen1981

J.P. Hansen and I.R. McDonald, Phys Rev A 23 2041 (1981)

Glosli2008(1,2)

J.N. Glosli et al. Phys Rev E 78 025401(R) (2008)

sarkas.potentials.qsp.Deutsch_force_P3M(r, pot_matrix)[source]¶

Calculate Deutsch QSP Force between two particles.

Parameters

  • r (float) – Distance between two particles.

  • pot_matrix (array) – It contains potential dependent variables. pot_matrix[0,:,:] = qi*qj/4*pi*eps0 pot_matrix[1,:,:] = 2pi/deBroglie pot_matrix[2,:,:] = e-e Pauli term factor pot_matrix[3,:,:] = e-e Pauli term exponent term pot_matrix[4,:,:] = Ewald parameter

Returns

  • U (float) – Potential.

  • force (float) – Force between two particles.
