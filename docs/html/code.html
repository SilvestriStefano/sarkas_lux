class S_integrator.Integrator(params, glb)[source]¶

Integrates the equation of motion for the particles. Updating position, velocity, and acceleration.

Verlet(ptcls)[source]¶

Update particle position and velocity based on velocity verlet method. More information can be found here: https://en.wikipedia.org/wiki/Verlet_integration or on the Sarkas website.

The velocity Verlet integration algorithm is comprised of a three-step update the following form:

  1. \(\mathbf{v}(t + \Delta t/2) = \mathbf{v}(t) + \mathbf{a}(t) \Delta t/2\)
  2. \(\mathbf{x}(t + \Delta t) = \mathbf{x}(t) + \mathbf{v}(t + \Delta t/2) \Delta t\)
  3. \(\mathbf{v}(t + \Delta t) = \mathbf{v}(t + \Delta t/2) + \mathbf{a}(t + \Delta t)\Delta t/2\)
