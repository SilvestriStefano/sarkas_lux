Sarkas Quickstart¶

Before you can begin working with Sarkas, you will need to have python3 installed. A useful python distribution that has many of the necessary packages for Sarkas is Anaconda. Anaconda can be downloaded here.

Obtaining Sarkas¶

Sarkas is undergoing heavy development but the most recent version can be found on GitHub, which can then be cloned.

Preliminary Packages¶

If you already have a version of python3 installed, you may still need to download additional packages such as pyfftw, fdint, and numba. If you have the most recent version of Anaconda installed, you should already have numba, installed. If you have the python package manager pip installed, you can install these packages, by simply opening a terminal and running the following commands:

To install pyfftw

$ pip install pyfftw

To install numba

$ pip install numba

To install fdint

$ pip install fdint

Testing Your Install¶

Once you have Sarkas and all necessary packages installed, you can run a test to make sure that all packages are installed correclty by navigating to the directory that contains Sarkas and running the command:

Using Sarkas¶

Input files¶

Unless adding new features to Sarkas such as new integrators, thermostats, potentials, etc, the only thing you will need to modify is the yaml.inp file. This file is responsible for specifying the simulation parameters such as the number of particles, number of timesteps, and initialization type for example.

Units¶

Currently, Sarkas uses Yukawa units to specify the system the user wants to simulate. For example, the user might want to model strongly coupled plasmas for a specific ion species and would need to supply the corresponding coulomb coupling paramters, \(\Gamma\), and electron screening parameter \(\kappa\). The coulomb coupling parameter for between species \(i\) and \(j\) is defined as

\[\Gamma_{ij} = \frac{Z_i Z_j e^2}{a_{ij} T_{ij}},\]

where \(Z_s\) is the effective charge for species \(s\), \(a_{ij} = (4 \pi n/3)^{-1/3}\) is the ion-sphere radius, \(n = n_i + n_j\) is the total particle number density, \(e\) is the elementary charge, and \(T_{ij} = (T_i + T_j)/2\) is the temperature of the system.

Additionally, the non-dimensional electron screening parameter is defined as

\[\kappa = \frac{a_{ij}}{\lambda_e},\]

where \(\lambda_e\) is the electron screening length defined as

\[\lambda_e^2 = \frac{\sqrt{ T_{ij} + \left(\frac{2}{3} E_F \right)^2 }}{4\pi n_e e^2}.\]

In the above expression, \(E_F\) is the Fermi energy, and \(n_e\) is the electron number density.
